# Security Policy

## Supported versions
Only the latest main branch is actively supported.

## Reporting a vulnerability
If you discover a security issue, don’t open a public issue.

Please report it privately by emailing:
webpro.aditya@gmail.com

Include:
- A clear description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggestions or fixes if you have any

We’ll review and respond as quickly as possible. Your report will be kept confidential until the issue is resolved.

## What we don’t accept
- Automated or vague reports with no detail
- Reports about dependencies that are already patched upstream
